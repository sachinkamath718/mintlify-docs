To set up a custom transform connector, please follow below steps:

1. Once you click on custom transform connector, you will be redirected to code editor page

![](/images/condense/v2.4.0/fundamentals/getting-set-up/rwKRGO3QthZ6EMqqYblg%2Fblobs%2FVgqDDEdf7ddLhDVytbi6%2FStep%25204A.png)

1. Click on “Connect to Github” to establish the connection between Github and Condense

2. After click on connect to github you shall be redirected to the github OAuth Page. Username and Password needs to be validated in the OAuth Page to successful sign-in to clone the repository to condense.

3. Once the sign-in is successful, you shall be r edirected back to the condense code editor page. If the sign-in is unsuccessful you will be redirected back to condense with OAuth authentication error.

4. If the sign-in was successful with the redirection you will be able to view the list of repositories from your github account which you have logged in to and your Github account username will reflect in code editor page.
