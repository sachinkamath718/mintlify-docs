# What's New in Condense v2.4.1

Condense v2.4.1 introduces enhancements across onboarding, organization management, billing, AI-powered platform operations, and product experience. This release improves the self-service evaluation journey, simplifies administration, introduces intelligent operational assistants, and enhances platform observability.

The following sections summarize the major capabilities introduced in this release.

## Try Condense for Free

The Try Condense for Free experience has been expanded with an end-to-end self-service onboarding and upgrade journey. Users can now evaluate Condense in a pre-configured environment and seamlessly transition to a production organization while continuing to use the same account. The enhanced workflow simplifies evaluation, reduces onboarding effort, and provides a more consistent upgrade experience.

## Multi-Organization Support

Users can now be members of multiple Condense organizations using a single account. This enhancement simplifies collaboration across different customers, projects, and environments while reducing the need to maintain multiple user accounts.

## Flexible Billing Models

Condense now supports both **Flat** and **Tiered** pricing models, enabling organizations to select a pricing structure that aligns with their operational and commercial requirements.

To ensure consistency across the platform, a centralized **Master Rate Card** has been introduced as the baseline for pricing across supported services and customer experiences.

## AI Agents 1.0

This release introduces the first version of **Condense AI Agents**, providing intelligent assistance for both platform operations and application development. The initial release includes specialized agents for Apache Kafka, Kubernetes, Grafana monitoring, and AI-assisted code generation.

## Platform Improvements

This release also includes stability improvements, usability enhancements, and quality fixes across the platform to improve reliability and the overall user experience.
