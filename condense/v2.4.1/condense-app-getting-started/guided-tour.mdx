# Guided Tour

The **Guided Tour** helps new users explore the Condense platform through an interactive walkthrough. It highlights the primary features and workflows, enabling users to quickly understand the platform without requiring prior knowledge.

The guided tour is automatically available for first-time users and can be restarted at any time from within the application.

![](/images/condense/v2.4.0/condense-app/guided-tour/image-1.png)

## Overview

The Guided Tour introduces users to the key areas of the Condense platform, including:

- Condense Console
- Condense Core
- Workspaces
- Connectors
- Transforms
- Pipelines
- Kafka Management
- Monitoring

Each step explains the purpose of the selected feature and provides contextual guidance for completing common tasks.

## Starting the Guided Tour

The Guided Tour starts automatically after signing in for the first time.

To restart the tour manually:

1. Sign in to Condense.
2. Open the required workspace.
3. Click the **Guided Tour** option.
4. Follow the on-screen instructions.

## Tour Controls

During the walkthrough, the following controls are available:

| Action | Description |
| --- | --- |
| Next | Moves to the next step in the tour. |
| Previous | Returns to the previous step. |
| Skip | Ends the current walkthrough. |
| Finish | Completes the guided tour. |

## Interactive Walkthrough

The Guided Tour highlights important user interface elements and explains their purpose, including:

- Navigation menu
- Workspace selection
- Pipeline designer
- Connector deployment
- Transform configuration
- Kafka Management
- Monitoring dashboards

Each step focuses on the current screen, allowing users to learn the platform progressively.

## Benefits

- Faster onboarding for new users.
- Reduced learning curve.

- Context-aware guidance.
- Consistent onboarding experience across the platform.

## Best Practices

- Complete the Guided Tour before creating your first pipeline.
- Explore each highlighted feature before moving to the next step.
- Restart the tour whenever you need a refresher after major platform updates.

## Frequently Asked Questions

### Does the Guided Tour start automatically?

Yes. The Guided Tour is displayed automatically for new users after their first sign-in.

### Can I skip the Guided Tour?

Yes. You can skip the tour at any time and restart it later if required.

### Can I restart the Guided Tour?

Yes. The Guided Tour can be launched again from within the Condense application.

### Does the Guided Tour make any configuration changes?

No. The Guided Tour is informational only and does not create or modify any resources.
