To choose output connector, please follow below steps:

1. Select “terminal option” option and click on “Next”

2. You will be redirected to connectors page where you can select output connector. Select appropriate output connector to add to your pipeline.

3. Click on “Add to Pipeline” and you will be redirected to “Configure Connector” page where you need to configure the input connector

4. Input all the mandatory fields for configuring the output connector.

5. Once the output connector is configured, deploy the output connector.

6. When the output connector is successfully deployed, it is available in workspace. The complete pipeline is available in workspace.
