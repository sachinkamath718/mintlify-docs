To select a custom transform, please follow below steps:

1. After configuring and deploying input connector, validate of the input connector is deployed successfully and running

2. To add custom transform to the input connector node, click on terminal option

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FGSIEPfeG8FzlnjLn6xvB%252FStep%25203A.png)

1. You will be redirected to connectors page where you can select application tab

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252F4hhXOXX3XZOjIsCTZUbG%252FStep%25203B.png)

1. Select Application connector to add to your pipeline
