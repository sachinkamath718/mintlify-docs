## 1. Telematics Device

To choose Telematics Device as an input connector, please follow below steps:

A. Select “Telematics Devices” option and click on “Next”

B. You will be redirected to list of Telematics devices to choose from along with search and filter functionality.

C. The list of devices may contain different models and different vendors to choose from as shown in the screenshot below.

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FP3mydpsbV1DBLiNK9VEp%252FStep%25202%2520C.png)

D. Click on “Add to Pipeline” and you will be redirected to “Configure Connector” page where you need to configure the input connector

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FHCH4dzgFxoe0LluWKFiw%252FStep%25202%2520C-1.png)

E. Once you configure the device, you shall be redirected to the node selection page where you can connect your device / connector to any other connector. As this device is of input data source no selection shall be available and default node is created.

F. Once the default node has been shown with all the configuration requirements you can click on deploy. Thus creating a new input node in your workspace.

G. To validate the status of your connector, check for the below symbols on your connectors: ![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FbkoQVGvLW72dQZvA3Ty0%252FConnected.svg) This symbol signifies that your connector is not successfully deployed ![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FA5AJJFl3DFM0EvTgJAb9%252FNot%2520Connected.svg) This symbol signifies that your connector is successfully deployed and running

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FSBnOBqJCMaRqSeXrrFwq%252FStep%25202%2520C-2.png)

## 2. Connectors as an input connector

To choose Connectors as an input connector, please follow below steps:

A. Select “Connectors” option and click on “Next”

B. You will be redirected to list of Connectors to choose from

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FkWAb9hxz9ogBDUgWj10h%252FStep%25202%2520D.png)

C. Click on “Add to Pipeline” and you will be redirected to “Configure Connector” page where you need to configure the input connector

D. Based on type of connector selected, you will have to input details for configuring the connector.

![](/images/condense/fundamentals/getting-set-up/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FHCH4dzgFxoe0LluWKFiw%252FStep%25202%2520C-1.png)
