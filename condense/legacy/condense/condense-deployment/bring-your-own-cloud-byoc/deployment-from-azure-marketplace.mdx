![](/images/condense/condense-deployment/bring-your-own-cloud-byoc/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FsOXZiNOEV8cslgJrqv68%252FAzure%25203.png)

### Step 4: The resource group that is mentioned needs to be a new one or an empty one with no resources in it.

![](/images/condense/condense-deployment/bring-your-own-cloud-byoc/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FvbacpakLuL4x2VBp2174%252Fuploads%252FCFuDGft9QF3lD2YHGtoX%252FAzure%25204.png)

### Step 5: By default, VPC, Subnet, Server size & Cluster size are mentioned, which are mandatory and can be changed by you. You will also have to provide the Kubernetes version and an admin email for communication.
