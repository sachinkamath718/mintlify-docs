## Highlights

| Category | What's New |
| --- | --- |
| **Getting Started** | **Try Condense for Free** introduces a self-service 30-day trial, enabling users to explore Condense with guided onboarding and access to platform capabilities without manual provisioning. |
| **User Experience** | **Guided Tour** provides an interactive walkthrough of the Condense interface, helping new users quickly become familiar with the platform and its core workflows. |
| **Kafka Management** | **Consumer Groups** and **Users & ACLs** extend Kafka Management with capabilities to monitor consumer groups and manage Kafka users and access permissions from within Condense. |
| **Application Management** | Improvements to application lifecycle management ensure applications remain intact when Git connectivity is temporarily unavailable, resulting in a more resilient deployment experience. |
| **Build & Deployment** | The **Docker Job Service** enhances application build workflows by improving how build jobs are executed within the platform. |
| **Stream Processing** | The **Alert** and **Split** utilities now support editing configured variables, simplifying the maintenance and update of existing stream processing pipelines. |
| **Connectors** | Added support for the **RIO OEM Connector**, expanding the range of supported telematics integrations. |
| **Security & Compliance** | Platform updates introduce enhancements aligned with **SOC2** and **GDPR** compliance requirements, strengthening security and privacy capabilities. |
| **Experience Center** | New **Demo Workspaces** provide guided, hands-on experiences that showcase Condense capabilities through interactive sample applications and use cases. |
